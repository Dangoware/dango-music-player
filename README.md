## Building
To build this project you'll need the [prerequisites](https://tauri.app/start/prerequisites/) for Tauri as well as Rust, Node, and npm.

### Clone the repository and install:

`cd dango-music-player && npm install`

### Dev

`cargo tauri dev`

### Build

`cargo tauri build`
